## 欢迎使用Hugo

这是一篇示例文章，展示如何在Hugo博客中编写内容。

## Markdown基础语法

### 标题

使用 `#` 符号创建标题，# 数量表示标题级别。

### 文本格式

- **粗体文本**：使用双星号包围
- *斜体文本*：使用单星号包围
- `代码片段`：使用反引号包围

### 列表

无序列表：
- 项目一
- 项目二
- 项目三

有序列表：
1. 第一步
2. 第二步
3. 第三步

### 代码块

```python
def hello_world():
    print("Hello, Hugo!")
    return True
```

```javascript
function greet(name) {
    console.log(`你好，${name}！`);
}
```
